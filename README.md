# New Pizza Town UI

A modern, responsive website for New Pizza Town restaurant featuring:

- Interactive menu system
- Online ordering
- Category navigation
- Customer reviews
- Location information
- Contact details

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts
